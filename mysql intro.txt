What is database? 

>> It is the collections of organized data, it can be easily maintained. 


Database management System(DBMS): A database management system is system software for creating and managing the databses. 


Types of data: 

  1. relational  (it shows the relation between the data, the relational data stored in the form of table.)

table: it is the combination of rows and columns.
 
  2. non-relational  (it doesn't show the relation between data, it stored in the form of file or folder)


RDBMS (Relation database management System): It is the type of Database management system where we can manage the relational databases.

Mysql: It is a relational database management system software, where we can manage the relational data, relational data is in the form of table. it supports the structured query language (sql).  

Command: 

1. show databases: for view all the databses list. 
2. create database databse_name: for creating the databse. 
3. drop database database_name: for dropping the database.
